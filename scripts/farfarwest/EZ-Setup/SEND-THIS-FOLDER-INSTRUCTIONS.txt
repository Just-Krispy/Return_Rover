Far Far West setup for Gavin

1. Right-click the zip and choose Extract All.
2. Open the extracted folder.
3. Double-click Run-This-First.cmd.
4. Click Yes if Windows asks for Administrator permission.
5. If the Put-Mod-Files-Here folder opens, put these 3 files in it:
   - XP-Gold-Souls
   - Better Wandering Traders
   - White Primary Pickup Glow
6. Go back to the black PowerShell window and press Enter.
7. Wait for PASS - All mod files installed.

Do not type paths. Do not paste instruction text into PowerShell.
If something turns red, send Ryan one screenshot.
